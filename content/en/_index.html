AMPLIFY API Management

{{< /blocks/section >}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-lightbulb" title="API Gateway" %}} API Gateway is a server-side application you can use to manage, deliver, and secure APIs. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="API Manager" %}} API Manager is a web-based API administration and management tool layered on API Gateway. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="API Portal" %}} API Portal is a self-service developer portal layered on both API Manager and API Gateway. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section color="primary" >}}

AMPLIFY Central

{{< /blocks/section >}} {{< blocks/section color="secondary">}} {{% blocks/feature icon="fa-lightbulb" title="API Management" %}} Design a custom API and import it into AMPLIFY Central. Add security to your API and share it with your consumers using AMPLIFY Catalog. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="DevOps" %}} Integrate AMPLIFY Central in your existing DevOps infrastructure using the AMPLIFY CLI or AMPLIFY Central DevOps API. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="Microservice Mesh Governance (BETA)" %}} Centrally manage APIs and microservices across multiple cloud and on-premise environments. {{% /blocks/feature %}} {{< /blocks/section >}}
